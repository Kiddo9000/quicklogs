A simple package for basic log messages. That's it.

npm install --save @crapintosh/quicklogs
Usage
const QuickLogs = require('@crapintosh/quicklogs'); QuickLogs.log('Log message'); QuickLogs.debug('Debug message'); QuickLogs.success('Success message'); QuickLogs.warning('Warning message'); QuickLogs.error('Error message'); QuickLogs.fatal('Fatal error message');
